Pickle

  •   Tue 02 October 2018
  •   tensorflow
  •   Pickle
  •   tensorflow python misc

It is often necessary to save some of the current status onto the hard disk and use it later. To save some status using pickle, there's nothing better than giving an example:

my_data_dict = {
    'a'     :   1,
    'b'     :   2,
    'my_obj':   <some_complex_object>
}
pickle_file = 'SOMEFILENAME.pickle'
if not os.path.isfile(pickle_file):
    print('Saving data to pickle file...')
    try:
        with open(pickle_file, 'wb') as pfile:
            pickle.dump(
                my_data_dict,
                pfile, pickle.HIGHEST_PROTOCOL)
    except Exception as e:
        print('Unable to save data to', pickle_file, ':', e)
        raise

and to restore:

pickle_file = 'SOMEFILENAME.pickle'
with open(pickle_file, 'rb') as f:
    pickle_data = pickle.load(f)
    a = pickle_data['a']
    b = pickle_data['b']
    my_obj = pickle_data['my_obj']
    del pickle_data  # free memory
